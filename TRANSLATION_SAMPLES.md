## Translation Notes

### Czech Translations
- **Formal register** used throughout for professional tone
- **Plural forms**: 3 forms (singular, 2-4, 5+)
- **Geographic terms**: Translated with precision and accuracy
- **Technical terms**: ISO codes and IDs kept in English

### German Translations
- **Formal "Sie"** form used consistently
- **Plural forms**: 2 forms (singular, plural)
- **Compound words**: Properly formed per German grammar rules
- **Professional tone**: Maintained throughout all content

### Intentionally NOT Translated
- ISO country codes (cca2, cca3)
- Wikidata QIDs
- API endpoints
- Internal database keys
- JavaScript variables
